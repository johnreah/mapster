# Non-functional requirements

## Tools and frameworks

- This is a Java GUI application built with the Java FX framework
- Aim to use the latest available version of Java (minimum 25)
- Aim to use the latest available version of JavaFX (minimum 25)

## Target environments

- The application should be able to build and run on Windows, MacOS and Linux
- It should be possible to build native executables for Windows and MacOS
